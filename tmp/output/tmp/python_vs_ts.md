# Python vs TypeScript 三个主要区别

## 1. 类型系统
- **Python**: 动态类型语言,变量类型在运行时确定,无需声明
- **TypeScript**: JavaScript 的超集,添加了静态类型系统,编译时检查类型

## 2. 运行环境
- **Python**: 通过解释器直接运行(.py 文件),有成熟的包管理工具 pip
- **TypeScript**: 需要先编译(transpile)成 JavaScript,然后在浏览器或 Node.js 中运行,使用 npm/yarn 管理依赖

## 3. 主要应用领域
- **Python**: 后端开发、数据科学、机器学习、自动化脚本、科学计算
- **TypeScript**: 前端开发(React/Vue/Angular)、全栈开发(Node.js)、大型 Web 应用

## 小结
Python 偏向数据处理和后端服务,类型灵活;TypeScript 偏向 Web 前端生态,类型严格且工具链丰富。
